# Stopwatch
A simple stopwatch app written in novus. My programming language. https://github.com/mjdaws0n/Novus. It uses the nox package manager https://github.com/mjdaws0n/Nox.

## Usage
```bash
stopwatch start
```
```bash
stopwatch stop
```

## Build
```bash
nox init
```
```bash
novus main.nov
```
